# 🏦 Python Banking System

A simple command-line Banking System built using Python. This project simulates basic banking operations such as balance inquiry, money deposit, withdrawal, and KYC management.

## 🚀 Features

* 💰 Check Account Balance
* 💵 Deposit Money
* 💸 Withdraw Money
* 📄 Check KYC Status
* 🆔 Update KYC Documents
* 🔁 Menu Driven Interface
* 🖥️ Simple Command Line Application

## 🛠️ Technologies Used

* Python 3
* Functions
* Dictionaries
* Loops
* Conditional Statements
* User Input Handling

## 📋 Available Operations

### 1. Check Balance

Displays the current account balance.

### 2. Deposit Money

Allows users to add money to their account.

### 3. Withdraw Money

Allows users to withdraw available funds.

### 4. Check KYC

Checks whether KYC documents have been submitted.

### 5. Update KYC

Allows users to submit document details for KYC verification.

### 6. Exit

Closes the application safely.
